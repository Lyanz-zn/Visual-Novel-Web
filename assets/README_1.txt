Taruh file gambar background di folder ini.

Format yang didukung: .jpg, .png, .webp

Nama file default yang dipakai di main.json:
  bg_intro.jpg   → background scene intro & scene kedua
  bg_menu.jpg    → background menu utama & about
  bg_study.jpg   → background scene belajar
  bg_quiz.jpg    → background scene quiz

Ukuran yang disarankan: 1920x1080px (16:9, landscape)

Jika file tidak ada, background akan menggunakan warna fallback
(biru gelap / #060a14) yang sudah didefinisikan di style.css.
